These programs scrape Google for a list of search queries. 
When there are new results from the scrapings, they are saved in a report.
Meant to keep track of when there is activity surrounding a search
query, like a stock, natural disaster etc. 
Evades bot detection by using cleared user agents and staggering requests.

Uses Python 3, Beautiful Soup 4 (bs4) and requests

The only difference between these two files is the directory 
delimeter \ vs /. 

On Linux will need to allow program to read and write.

Can easily be set up to run on cron job.
